# 产业链深度分析模型演示指南

## 快速开始

### 1. 启动系统

```bash
# 在项目目录下启动HTTP服务器
python3 -m http.server 8080

# 或使用Node.js
npx http-server -p 8080
```

然后在浏览器中访问：`http://localhost:8080`

### 2. 文件结构

```
tech-algorithms-prototype/
├── index.html              # 主页面（产业链深度分析模型）
├── industrial-chain.js     # 13个子功能配置数据
├── index.html.backup       # 原始备份文件
├── docs/
│   └── industrial_chain.md # 需求文档
├── speech.md               # 演讲稿
├── demo-guide.md           # 本文件（演示指南）
└── tests/
    ├── manual-test.js      # 手动测试脚本
    └── basic.spec.ts       # Playwright测试文件
```

## 功能概览

### 13个核心功能

1. **产业链全景模型** - 树状图展示产业链上下游关系
2. **深圳产业链模型** - 表格展示深圳各区域产业分布
3. **产业发展路线历程模型** - 时间线展示重大事件
4. **产业热点领域识别模型** - 热力图展示热点排名
5. **深圳产业优势识别模型** - 多维度对比分析
6. **产业周期分析模型** - 表格展示产业周期阶段
7. **产业阶段分析模型** - 进度条展示成熟度
8. **产业链关键环节识别模型** - 表格展示关键节点
9. **产业瓶颈堵点识别模型** - 分类展示瓶颈问题
10. **区域产业指数模型** - 表格展示区域对比
11. **深圳问题识别模型** - 分类展示问题短板
12. **深圳机遇分析模型** - 卡片展示发展机遇
13. **重点布局产业推荐模型** - 表格+卡片展示推荐

## 演示流程建议

### 方案一：全面演示（10-15分钟）

适合详细展示给技术评审或专家团队。

1. **开场介绍**（1分钟）
   - 系统定位和目标
   - 13个功能模块概述

2. **产业链全景模型**（2分钟）
   - 展示树状图结构
   - 点击"执行分析"按钮
   - 展示产业链层级关系

3. **深圳产业链模型**（1分钟）
   - 展示深圳各区域分布
   - 高亮核心区域南山区

4. **产业发展路线历程模型**（1分钟）
   - 展示时间线布局
   - 强调重大事件标注

5. **产业热点领域识别模型**（1分钟）
   - 展示热力图
   - 强调高分热点（生成式AI、新能源等）

6. **深圳产业优势识别模型**（1分钟）
   - 展示多维度对比
   - 强调TOP3产业

7. **产业瓶颈堵点识别模型**（1分钟）
   - 展示分类瓶颈
   - 强调严重程度标注

8. **重点布局产业推荐模型**（1分钟）
   - 展示推荐列表
   - 强调TOP5产业

9. **总结交流**（2分钟）
   - 系统特点总结
   - Q&A环节

适合向领导或非技术人员快速展示核心价值。

   - 强调数据驱动决策价值

### 方案三：深度技术演示（20-30分钟）

适合向技术团队详细介绍实现细节。

1. **技术架构介绍**（5分钟）

   - 数据结构设计
   - 可视化方案

2. **各功能详细演示**（15-20分钟）
   - 逐个展示13个功能

## 演示技巧

### 1. 突出重点

- **重点功能**：产业链全景模型、热点识别、重点布局推荐
- **关键数据**：TOP排名、高热度、严重程度
- **可视化亮点**：树状图、热力图、时间线

### 2. 交互演示

- 点击"执行分析"按钮展示加载动画
- 点击复选框展示交互效果
- 切换不同模块展示页面动画

### 3. 数据讲解

- 解释数据来源和含义
- 说明指标计算方式
- 对比分析结果意义

### 4. 故事化演示

以具体产业（如半导体、AI）为主线，串联各个功能：

- "我们先看半导体产业的全貌"
- "再看深圳的布局情况"
- "然后分析发展历程"
- "识别热点和瓶颈"
- "最后给出布局建议"

## 常见问题解答

### Q1: 数据是真实的吗？

A: 当前原型使用模拟数据，真实系统将对接深圳市科技创新服务平台的数据中枢。

### Q2: 可以自定义产业吗？

A: 是的，左侧输入面板可以自定义选择产业、时间范围等参数。

### Q3: 如何导出分析结果？

A: 点击"导出结果"按钮，可以导出CSV、Excel等格式的分析报告。

### Q4: 系统支持移动端吗？

A: 系统采用响应式设计，支持PC端和移动端访问。

### Q5: 如何对接真实数据？

A: 通过API接口对接数据融合中枢，替换industrial-chain.js中的模拟数据。
